## 1. 使用者管理

管理員擁有最高權限，可新增與管理所有帳號。

### 1.1 新增使用者
- 進入「使用者管理」介面。
- 點擊「新增使用者」。
- 填寫姓名、Email、角色 (Admin/Teacher/Student) 及預設密碼。
- 設定帳號狀態為 `Active` 以立即啟用。

### 1.2 編輯與重設密碼
- 在使用者列表中找到特定使用者。
- 點擊「編輯」可修改基本資料。
- 點擊「重設密碼」可發送重設連結或直接設定新密碼。

### 1.3 批次匯入
- 支援 CSV 格式匯入學生與教師名單。
- 請下載系統提供的範本，填入資料後上傳。

## 2. 班級與課程設定

### 2.1 建立學期與班級
1. 進入「學期設定」，新增當前學年度與學期。
2. 進入「班級管理」，建立如 "113-1 資工一甲" 的班級群組。

### 2.2 開設課程
1. 進入「課程管理」，點擊「新增課程」。
2. 設定課程名稱、授課教師、所屬班級。
3. 設定**時間表 (Schedule)**：每週固定時段或特定日期。

## 3. 資料維護

### 3.1 資料庫操作 (Prisma Studio)
若需直接檢視或修改資料庫內容，可使用 Prisma Studio：
```bash
# 在後端目錄執行
npx prisma studio
```
瀏覽器將開啟 `http://localhost:5555`，可直接編輯所有資料表。

### 3.2 系統備份
- 系統建議每日進行資料庫備份。
- 備份指令 (Docker 環境):
  ```bash
  docker exec -t student_202511-postgres-1 pg_dumpall -c -U postgres1 > dump_`date +%d-%m-%Y"_"%H_%M_%S`.sql
  ```

## 4. 報表功能

### 4.1 全校出勤統計
- 儀表板提供全校出勤概況圖表。
- 可篩選特定日期區間下載 Excel 報表。

### 4.2 異常出勤追蹤
- 系統會自動標記連續缺席的學生。
- 管理員可匯出「異常出勤名單」以供輔導室追蹤。
