## 1. 登入與帳號

### 登入系統
1. 開啟系統網址（預設為 `http://localhost:3000`）。
2. 在登入頁面輸入您的**電子郵件**與**密碼**。
   - **教師預設帳號**: `teacher@example.com` / `teacher123`
   - **學生預設帳號**: `student@example.com` / `student123`
3. 點擊「登入」按鈕。

### 忘記密碼
若您忘記密碼，請點擊登入頁面的「忘記密碼」連結，輸入您的註冊信箱，系統將發送重設連結給您。若無法使用信箱，請聯繫系統管理員。

## 2. 教師功能

教師主要負責管理課程與記錄學生出勤。

### 2.1 課程管理
- 登入後，在儀表板可看見您負責的所有課程。
- 點擊特定課程可查看詳細資訊、學生名單與課程公告。

### 2.2 點名與出勤記錄
1. 進入課程頁面，選擇「點名」或「出勤記錄」。
2. 系統列出該堂課的學生名單。
3. 您可以將學生狀態標記為：
   - **出席 (Present)**
   - **遲到 (Late)**
   - **缺席 (Absent)**
   - **請假 (Leave)**
4. 完成後點擊「儲存」。

### 2.3 請假審核
- 當學生提出請假申請時，您會收到通知。
- 在「待審核申請」區域，您可以查看請假事由並選擇「核准」或「駁回」。

## 3. 學生功能

學生可查看課表、出勤狀況並申請請假。

### 3.1 查看課表
- 登入後，儀表板會顯示本週課表。
- 點擊課程可查看授課教師與教室資訊。

### 3.2 查詢出勤紀錄
- 點擊「出勤紀錄」選單。
- 您可以查看所有課程的出席統計（如：出席率 95%）。
- 點擊特定課程可查看每日詳細出勤狀況。

### 3.3 線上請假
1. 點擊「請假申請」。
2. 選擇請假日期與課程場次。
3. 填寫請假類別（如：病假、事假）與事由。
4. 提交申請。
   - 申請狀態將顯示為「待審核」。
   - 教師核准後，狀態將變更為「已核准」，並自動更新出勤紀錄。

## 4. 常見問題

**Q: 登入時顯示「帳號尚未啟用」？**
A: 請聯繫系統管理員開通您的帳號。

**Q: 如何修改個人資料或密碼？**
A: 點擊右上角的人像圖示，選擇「個人設定」即可修改密碼與聯絡資訊。

**Q: 請假申請送出後可以取消嗎？**
A: 在教師尚未審核前，您可以進入申請紀錄點擊「撤回」。若已審核，需請教師協助修改。
